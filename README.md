# 🛒 "Reducing Delivery delays and improving customer satisfaction in E-Commerce Operations".

## 📌 Project Overview

This project is an end-to-end Data Analytics solution developed using **SQL, Python, and Power BI** on the **Olist Brazilian E-Commerce Dataset**.

The primary objective of this project is to analyze customer purchasing behavior, sales performance, payment trends, delivery efficiency, and product-level insights to generate actionable business intelligence and support data-driven decision-making.

This project demonstrates the complete analytics workflow followed by Data Analysts in real-world business environments — from raw data processing to dashboard storytelling and customer segmentation.

# 💡 Key Business Insights

- Revenue demonstrated strong growth throughout 2017 and 2018.
- Credit cards dominated transaction payments across the platform.
- Certain product categories generated significantly higher revenue contribution.
- Delivery delays impacted customer satisfaction and fulfillment efficiency.
- Monthly revenue trends indicated strong marketplace expansion.
